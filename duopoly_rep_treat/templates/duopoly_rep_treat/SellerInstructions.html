Imagine that you are playing as a seller.

Each round, you will be placed into a group with {{ Constants.buyers_per_group }} buyers and {{ Constants.sellers_per_group|add:-1 }} other seller{{ Constants.sellers_per_group|add:-1 | pluralize }}.

You can produce and sell objects to earn tokens. You do not value owning objects themselves, but you can sell objects to buyers. As a seller you first choose the prices at which you will sell objects in the market. In this game, each object has {{ subsession.dims }} price{{ subsession.dims |pluralize }}. All {{ subsession.dims }} price{{ subsession.dims |pluralize }} are for a single object, and each object you sell will earn you the sum of the {{ subsession.dims }} price{{ subsession.dims |pluralize }}.

After you set your prices, the buyers will decide if they want to buy from you or from the other seller. You will then automatically produce the number of objects you sold in the market.

Your cost of production is {{ Constants.prodcost }} tokens per objects. So,

  • If you sell zero objects, you make 0 tokens and spend 0 tokens.
  • If you sell one object, you make the sum of your {{ subsession.dims }} price{{ subsession.dims |pluralize }} and spend {{ Constants.prodcost }} tokens.
  • If you sell two objects, you make two times the sum of your {{ subsession.dims }} price{{ subsession.dims |pluralize }} and spend {{ Constants.prodcost|add:Constants.prodcost }} tokens.
